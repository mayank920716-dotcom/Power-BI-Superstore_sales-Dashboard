# 📊 Superstore Sales Dashboard (Power BI)

## 📌 Project Overview

This project presents an interactive **Sales Dashboard** built using Power BI to analyze the Superstore dataset. The dashboard provides meaningful insights into sales performance, profit distribution, and customer ordering patterns.

It helps stakeholders make data-driven decisions by visualizing key business metrics in a clear and professional format.

---

## 🎯 Objectives

* Analyze overall sales and profit performance
* Track monthly sales trends
* Identify top-performing regions and categories
* Understand order patterns and average sales

---

## 📈 Key Insights & Features

* **Total Sales** – Overall revenue generated
* **Total Profit** – Net profit from all sales
* **Total Orders** – Number of unique orders
* **Average Sales** – Average revenue per order

### 📊 Visualizations Included:

* Sales Trend (Monthly Line Chart)
* Profit by Category (Bar Chart)
* Sales by Region (Bar Chart)
* Top Orders Table
* Interactive Filters (Region, Category, Order Date)
